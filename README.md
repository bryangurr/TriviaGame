## Installation & Setup

1. **Clone this repository:**
   ```bash
   git clone https://github.com/bryangurr/TriviaGame.git
   cd TriviaGame
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser to:
   - http://localhost:3000
